من أين أحصل على الملفات؟ (مصادر موثوقة ونصائح)
- ابحث أولاً عن الصفحات الرسمية لمصممي الخطوط أو مكتبات الخطوط المفتوحة المصدر على GitHub.
- مواقع خطوط مجانية/مفتوحة المصدر: Font Squirrel (https://www.fontsquirrel.com), Font Library (https://fontlibrary.org).
- إذا كان الخط متاحًا على Google Fonts فاختره من هناك (جرب البحث "KAFA font" أو "DAMAS SANA font" مع إضافة "woff2" لنتائج أفضل).
- كن حذرًا من مواقع التحميل غير الموثوقة؛ تحقق من ترخيص الاستخدام (Commercial / SIL / OFL) قبل الاستخدام في مشروع حي.

إرشادات تقنية:
- الأفضل استخدام صيغة .woff2 للويب لأنها أصغر وأسرع تحميلًا.
- إذا لم تجد .woff2، يمكنك استخدام .woff أو .ttf ثم تحويلها إلى .woff2 عبر أدوات موثوقة (مثل transfonter.org).
- بعد رفع الملفات داخل هذا المجلد، لا حاجة لتعديل CSS إذا حافظت على نفس أسماء الملفات؛ وإلا عدّل `url()` في [css/style.css](css/style.css) للإشارة إلى المسارات الصحيحة.

بدائل إن لم تجد الخطوط المطلوبة:
